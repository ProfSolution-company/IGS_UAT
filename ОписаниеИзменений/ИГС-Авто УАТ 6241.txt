Внешняя обработка "Пакетная выписка путевых листов (УАТ)"

Реквизиты обработки:
	Добавлены новые реквизиты:
		Дата - новый реквизит, тип СтандартныйПериод
		ЗаказНаТС - новый реквизит, тип "ДокументСсылка.уатЗаказГрузоотправителя"
		ДвеСмены - новый реквизит, тип Булево
		Водитель - новый реквизит, тип "СправочникСсылка.Сотрудники"
		Водитель2 - новый реквизит, тип "СправочникСсылка.Сотрудники"
		Смена - новый реквизит, тип "СправочникСсылка.уатСмены"
		Смена2 - новый реквизит, тип "СправочникСсылка.уатСмены"

Модуль объекта:
	СведенияОВнешнейОбработке() - описание внешней обработки
	
Модуль формы:
	ПриСозданииНаСервере() - доработана процедура, обработка видимости "Смены2" и заполнение стандартного периода
	ЗаполнитьТаблицуТС() - новая процедура заполнения таблицы значений на форме
    ЗаполнитьСписокТС() - новая процедура команды "ЗаполнитьСписокТС"
    ЗаказНаТСПриИзменении() - новая процедура заполнения таблицы значений на форме при изменении поля ввода "ЗаказНаТС"
    СформироватьПодготовкуСервер() - процедура переделана для заполнения изменненной таблицы значений "Подготовка" на форме
    ДвеСменыПриИзменении() - новая процедура для управления заголовком поле ввода "Смена" и видимостью поля ввода "Смена2"
    
Форма:
	ТС - новая таблица значений, заполняемая по команде "Заполнить список ТС" и при изменении поля ввода "ЗаказНаТС"
	Подготовка - переделанная таблица значений под задачи проекта
	Настройки - закладка  изменена согласно проекта
